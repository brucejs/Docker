# php-cachet

Custom PHP image based upon php:7.1-fpm for running [Cachet v2.3.18](https://github.com/CachetHQ/Cachet/tree/v2.3.18).

Required PHP modules:

- apcu
- gd
- mcrypt
- mysqli
- pdo_mysql
- pdo_pgsql
- pdo_sqlite<sup>\*</sup>
- pgsql
- redis
- simplexml<sup>\*</sup>
- sqlite3<sup>\*</sup>
- tokenizer<sup>\*</sup>
- xml<sup>\*</sup>

\*<sup>already included in php:7.1-fpm</sup>
